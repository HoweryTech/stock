# 执行闭环报告示例

本文档展示执行闭环总检查报告的典型结构。示例内容不是买卖建议，也不是实际交易记录。

```text
# 执行闭环总检查

- 决策边界：本报告只汇总执行、卖出和复盘记录的规则检查结果，不构成买卖建议。
- 总结论：needs_review
- 阻断项记录数：0
- 需复核记录数：3
- 缺失下游记录数：2
- 孤儿记录数：1

## 缺失下游记录

- [missing_position_from_trade_execution] 买入执行 EXEC-20260710-0001 已通过检查，但未找到来源执行对应的持仓记录。
  - fix: 运行 tools/new_position_from_execution.py --execution <executions/EXEC-20260710-0001.yaml> 生成持仓记录。
- [missing_review_from_exit_execution] 卖出执行 EXITEXEC-20260710-0001 已通过检查，但未找到来源卖出执行对应的复盘记录。
  - fix: 运行 tools/new_trade_review_from_exit_execution.py --exit-execution <exit-executions/EXITEXEC-20260710-0001.yaml> 生成复盘草稿。

## 孤儿记录

- [position_source_execution_not_found] 持仓 POS-20260710-0001 的来源买入执行 EXEC-MISSING 不在执行记录中。
  - fix: 补回来源买入执行记录，或修正持仓的 execution_snapshot.execution.id。

## 修复动作分组

- 生成持仓记录：1 项
  - [missing_position_from_trade_execution] EXEC-20260710-0001：买入执行 EXEC-20260710-0001 已通过检查，但未找到来源执行对应的持仓记录。
    - fix: 运行 tools/new_position_from_execution.py --execution <executions/EXEC-20260710-0001.yaml> 生成持仓记录。
- 生成复盘记录：1 项
  - [missing_review_from_exit_execution] EXITEXEC-20260710-0001：卖出执行 EXITEXEC-20260710-0001 已通过检查，但未找到来源卖出执行对应的复盘记录。
    - fix: 运行 tools/new_trade_review_from_exit_execution.py --exit-execution <exit-executions/EXITEXEC-20260710-0001.yaml> 生成复盘草稿。
- 修正来源引用：1 项
  - [position_source_execution_not_found] POS-20260710-0001：持仓 POS-20260710-0001 的来源买入执行 EXEC-MISSING 不在执行记录中。
    - fix: 补回来源买入执行记录，或修正持仓的 execution_snapshot.execution.id。

## 买入执行记录

- EXEC-20260710-0001 conclusion=pass blockers=0 warnings=0 path=executions/EXEC-20260710-0001.yaml

## 卖出执行记录

- EXITEXEC-20260710-0001 conclusion=pass blockers=0 warnings=0 path=exit-executions/EXITEXEC-20260710-0001.yaml

## 交易复盘记录

- TR-20260710-0001 conclusion=pass blockers=0 warnings=0 path=reviews/TR-20260710-0001.yaml
```

阅读顺序建议：

1. 先看 `总结论`，确认是否存在阻断项。
2. 再看 `修复动作分组`，按同类问题批量处理。
3. 最后看各记录明细，确认具体文件路径和检查结论。
