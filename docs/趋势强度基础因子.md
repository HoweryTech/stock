# 趋势强度基础因子

本文档定义第一版趋势强度因子的计算规则。

当前阶段只计算简单、可解释、可复核的基础因子，不引入复杂模型。后续趋势筛选器会读取这些因子，结合投资体系配置生成候选股。

## 1. 输入数据

基础因子计算依赖：

- `data/processed/daily_bars.csv`：标准化后的日线行情。
- `data/processed/tradable_universe.csv`：可选，可交易股票池。

如果提供可交易股票池，则只计算股票池内标的；否则对日线行情中的全部股票计算。

## 2. 输出数据

默认输出：

```text
data/processed/trend_factors.csv
data/metadata/trend_factors.json
```

输出是一份最新截面数据，每只股票一行。

## 3. 计算字段

基础字段：

| 字段 | 说明 |
| --- | --- |
| code | 股票代码 |
| trade_date | 最新交易日期 |
| close | 最新收盘价 |
| bars_count | 可用日线数量 |
| is_suspended | 最新交易日是否停牌 |
| is_limit_up | 最新交易日是否涨停 |
| is_limit_down | 最新交易日是否跌停 |

窗口字段按 `--windows` 参数生成。例如 `--windows 2,5,20` 会生成：

| 字段 | 说明 |
| --- | --- |
| return_2d | 最近 2 根日线区间收益率 |
| ma_2 | 最近 2 根日线收盘价均值 |
| above_ma_2 | 最新收盘价是否高于 MA2 |
| turnover_avg_2 | 最近 2 根日线平均成交额 |

其他窗口同理。

## 4. 计算命令

```bash
python3 tools/calc_trend_factors.py \
  --daily-bars data/processed/daily_bars.csv \
  --universe data/processed/tradable_universe.csv \
  --output data/processed/trend_factors.csv \
  --metadata-output data/metadata/trend_factors.json \
  --windows 5,20
```

使用样例数据：

```bash
python3 tools/calc_trend_factors.py \
  --daily-bars samples/daily_bars.sample.csv \
  --output /tmp/trend_factors.csv \
  --metadata-output /tmp/trend_factors.json \
  --windows 2
```

## 5. 设计原则

- 因子必须可解释。
- 输入和输出都使用 CSV，方便检查。
- 不足窗口的数据保留空值，不伪造结果。
- 每次计算输出元数据，记录输入、窗口、股票数量和日期范围。
