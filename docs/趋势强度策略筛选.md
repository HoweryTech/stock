# 趋势强度策略筛选

本文档定义第一版趋势强度策略筛选器。

筛选器读取趋势强度基础因子，输出候选股、入选原因和风险提示。它不直接给出买入建议，只负责判断股票是否进入观察池。

## 1. 输入数据

默认输入：

```text
data/processed/trend_factors.csv
```

该文件由 `tools/calc_trend_factors.py` 生成。

## 2. 策略配置

默认配置位于：

```text
config/investment-profile.example.yaml
```

配置路径：

```text
strategies.trend_strength.screening
```

第一版字段：

| 字段 | 说明 |
| --- | --- |
| window | 使用哪个窗口，例如 2、5、20 |
| min_return_pct | 最低区间收益率 |
| require_above_ma | 是否要求最新收盘价站上均线 |
| exclude_suspended | 是否排除停牌 |
| exclude_limit_up | 是否排除涨停 |
| exclude_limit_down | 是否排除跌停 |
| max_candidates | 最多输出候选数 |

## 3. 输出数据

默认输出：

```text
data/processed/trend_candidates.csv
data/metadata/trend_candidates.json
```

候选股字段：

```text
code,trade_date,score,close,return,ma,above_ma,turnover_avg,reasons,risks
```

## 4. 筛选命令

```bash
python3 tools/screen_trend_strength.py \
  --profile config/investment-profile.example.yaml \
  --factors data/processed/trend_factors.csv \
  --output data/processed/trend_candidates.csv \
  --metadata-output data/metadata/trend_candidates.json
```

使用样例数据：

```bash
python3 tools/calc_trend_factors.py \
  --daily-bars samples/daily_bars.sample.csv \
  --output /tmp/trend_factors.csv \
  --metadata-output /tmp/trend_factors.json \
  --windows 2

python3 tools/screen_trend_strength.py \
  --factors /tmp/trend_factors.csv \
  --output /tmp/trend_candidates.csv \
  --metadata-output /tmp/trend_candidates.json
```

## 5. 设计原则

- 输出观察池，不输出买入结论。
- 每个候选必须有入选原因。
- 风险提示要保留，尤其是涨停、跌停、停牌、数据不足。
- 排序规则保持简单可解释，第一版按收益率和均线状态打分。
