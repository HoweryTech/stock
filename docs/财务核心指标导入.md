# 财务核心指标导入

本文档定义财务核心指标的真实抓取、CSV 标准结构和导入规则。

当前已支持通过东方财富公开财务接口抓取全市场财务指标，同时保留本地 CSV 导入入口。后续接入 AKShare、Tushare、BaoStock 或其他数据源时，也应先转换为本文档定义的标准结构，再进入价值质量策略、风险过滤和候选股解释模块。

## 1. 目录约定

```text
data/
  raw/snapshots/financial_metrics/YYYY-MM-DD/
                每次抓取后留存的财务指标快照
  processed/    标准化后的财务指标
  metadata/     导入元数据和质量报告
```

真实财务数据默认不提交到 Git。

## 2. 财务核心指标 CSV

第一版财务指标文件至少包含以下字段：

| 字段 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| report_period | 是 | date | 报告期，格式 YYYY-MM-DD |
| code | 是 | string | 股票代码，保留前导 0 |
| roe | 否 | number | 净资产收益率，百分比 |
| roa | 否 | number | 总资产收益率，百分比 |
| gross_margin | 否 | number | 毛利率，百分比 |
| net_margin | 否 | number | 净利率，百分比 |
| debt_ratio | 否 | number | 资产负债率，百分比 |
| operating_cash_flow | 否 | number | 经营现金流净额，元 |
| revenue_growth_yoy | 否 | number | 营收同比，百分比 |
| net_profit_growth_yoy | 否 | number | 净利润同比，百分比 |
| deducted_net_profit_growth_yoy | 否 | number | 扣非净利润同比，百分比 |
| eps | 否 | number | 每股收益 |
| data_source | 否 | string | 数据来源 |
| updated_at | 否 | date | 数据更新时间，格式 YYYY-MM-DD |

## 3. 真实抓取命令

首次建库建议拉取近 3-5 年报告期，按 `code + report_period` 合并去重：

```bash
.venv/bin/python tools/fetch_eastmoney_financial_metrics.py \
  --codes-file data/processed/tradable_universe.csv \
  --report-count 20 \
  --output data/processed/financial_metrics.csv \
  --metadata-output data/metadata/financial_metrics.fetch.json \
  --workers 4
```

日常增量只拉最近报告期即可：

```bash
.venv/bin/python tools/fetch_eastmoney_financial_metrics.py \
  --codes-file data/processed/tradable_universe.csv \
  --report-count 1 \
  --output data/processed/financial_metrics.csv \
  --metadata-output data/metadata/financial_metrics.fetch.json \
  --workers 4
```

抓取工具会执行：

- 按股票代码抓取东方财富最新财务报告。
- 转换为标准财务字段。
- 与已有 `data/processed/financial_metrics.csv` 按 `code + report_period` 合并去重。
- 输出抓取元数据和失败代码列表。
- 将当前标准化结果留存到 `data/raw/snapshots/financial_metrics`。

## 4. 本地导入命令

```bash
python3 tools/import_financial_metrics.py \
  --input samples/financial_metrics.sample.csv \
  --output data/processed/financial_metrics.csv \
  --metadata-output data/metadata/financial_metrics.import.json
```

导入工具会执行：

- 必填字段检查。
- 报告期校验。
- 股票代码格式校验。
- 数值字段标准化。
- 同一股票同一报告期重复记录检查。
- 标准 CSV 输出。
- 导入元数据输出。

## 5. 输出标准

标准输出 CSV 字段顺序固定为：

```text
report_period,code,roe,roa,gross_margin,net_margin,debt_ratio,operating_cash_flow,revenue_growth_yoy,net_profit_growth_yoy,deducted_net_profit_growth_yoy,eps,data_source,updated_at
```

后续价值质量策略会读取 `data/processed/financial_metrics.csv`。
