# 组合持仓日检

本文档定义多个持仓的组合级日检汇总。

单个持仓日检用于更新价格并检查个股风险；组合持仓日检用于汇总所有持仓的组合风险。

## 1. 输入

默认读取：

```text
positions/*.yaml
config/investment-profile.example.yaml
```

可以传入多个文件或 glob：

```bash
python3 tools/check_portfolio_positions.py \
  --positions positions/*.yaml
```

## 2. 汇总内容

工具会输出：

- 持仓数量。
- 总仓位。
- 行业仓位。
- 需要处理的持仓数量。
- 有提醒的持仓数量。
- 组合层面的仓位超限或接近上限提醒。
- 每个持仓的 `position_check` 结果。

## 3. 命令

```bash
python3 tools/check_portfolio_positions.py \
  --positions positions/*.yaml \
  --output data/metadata/portfolio_positions.check.json
```

输出 JSON：

```bash
python3 tools/check_portfolio_positions.py \
  --positions positions/*.yaml \
  --json
```

## 4. 结论

- `normal`：组合和单票检查均正常。
- `warning`：存在提醒项。
- `needs_action`：存在需要处理的持仓或组合仓位超限。
