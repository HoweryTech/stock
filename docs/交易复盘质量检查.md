# 交易复盘质量检查

本文档定义交易复盘草稿的完整性和归因质量检查。

复盘质量检查不判断策略未来是否有效，只判断这份复盘是否能解释：为什么买、为什么卖、是否按计划、收益如何、下一步怎么改。

## 1. 输入

默认读取：

```text
reviews/*.yaml
```

## 2. 阻断项

- 缺少复盘编号、来源交易计划、股票信息。
- 缺少入场日期、退出日期、入场价、退出价、仓位。
- 缺少退出原因、单笔收益率、组合收益贡献、结果分类。
- 入场价、退出价或仓位不大于 0。
- 结果分类不在允许范围内。

阻断项表示这份复盘不能用于统计归因。

## 3. 提醒项

- 收益率和入场/退出价计算不一致。
- 组合收益贡献和仓位/收益率计算不一致。
- 执行错误类复盘缺少错误标签。
- 未回答是否符合计划、买入理由是否仍成立、风控是否执行、仓位是否执行。
- 复盘教训或下一步动作为空。

## 4. 命令

```bash
python3 tools/check_trade_review_quality.py \
  --review reviews/TR-示例.yaml
```

输出 JSON：

```bash
python3 tools/check_trade_review_quality.py \
  --review reviews/TR-示例.yaml \
  --json
```
