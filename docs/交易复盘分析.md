# 交易复盘分析

本文档定义如何从复盘记录生成策略迭代分析。

复盘分析不预测未来收益，也不自动修改策略参数。它只汇总已经发生的交易结果，帮助识别哪些策略、错误标签和复盘质量问题需要关注。

## 1. 输入

默认读取：

```text
reviews/*.yaml
```

## 2. 输出

默认输出：

```text
reports/review-analysis.md
```

可选输出 JSON：

```text
data/metadata/review-analysis.json
```

真实分析报告默认不提交到 Git。

## 3. 统计内容

- 总复盘数量。
- 胜率。
- 平均单笔收益率。
- 组合收益贡献合计。
- 按策略汇总胜率、收益和组合贡献。
- 结果分类分布。
- 错误标签分布。
- 复盘质量分布。

## 4. 命令

```bash
python3 tools/analyze_trade_reviews.py \
  --reviews reviews/*.yaml \
  --output reports/review-analysis.md \
  --json-output data/metadata/review-analysis.json
```

如果只想看 JSON：

```bash
python3 tools/analyze_trade_reviews.py --json
```
