# 日线行情数据导入

本文档定义第一阶段日线行情 CSV 的标准结构和导入规则。

当前支持两种入口：

- 使用 `tools/fetch_daily_bars_sina.py` 自动拉取公开日线行情，并写入本文档定义的标准结构。
- 使用本地 CSV 导入。后续接入 AKShare、Tushare、BaoStock 或 yfinance 时，也应先转换为本文档定义的标准结构，再进入因子计算、策略筛选和回测模块。

## 1. 目录约定

```text
data/
  raw/          原始日线行情文件
  processed/    标准化后的日线行情
  metadata/     导入元数据和质量报告
```

真实行情数据默认不提交到 Git。

## 2. 日线行情 CSV

第一版日线行情文件至少包含以下字段：

| 字段 | 必填 | 类型 | 说明 |
| --- | --- | --- | --- |
| trade_date | 是 | date | 交易日期，格式 YYYY-MM-DD |
| code | 是 | string | 股票代码，保留前导 0 |
| open | 是 | number | 开盘价 |
| high | 是 | number | 最高价 |
| low | 是 | number | 最低价 |
| close | 是 | number | 收盘价 |
| pre_close | 否 | number | 前收盘价 |
| volume | 是 | number | 成交量 |
| turnover | 是 | number | 成交额 |
| turnover_rate | 否 | number | 换手率，百分比 |
| is_limit_up | 是 | bool | 是否涨停 |
| is_limit_down | 是 | bool | 是否跌停 |
| is_suspended | 是 | bool | 是否停牌 |
| adjust_type | 否 | string | 复权类型：none、qfq、hfq |
| data_source | 否 | string | 数据来源 |
| updated_at | 否 | date | 数据更新时间，格式 YYYY-MM-DD |

布尔字段支持：

```text
true / false
1 / 0
yes / no
y / n
是 / 否
```

## 3. 导入命令

自动拉取指定股票：

```bash
python3 tools/fetch_daily_bars_sina.py \
  --codes 600000 300750 \
  --output data/processed/daily_bars.csv \
  --metadata-output data/metadata/daily_bars.fetch.json
```

从持仓文件自动拉取：

```bash
python3 tools/fetch_daily_bars_sina.py \
  --positions positions/*.yaml \
  --output data/processed/daily_bars.csv \
  --metadata-output data/metadata/daily_bars.fetch.json
```

拉取工具会将新浪日 K 转换为项目标准字段。新浪日 K 不直接提供成交额时，工具用 `close * volume` 估算 `turnover`，因此该字段适合做流动性近似筛查，不适合精确成交额统计。

手工 CSV 导入：

```bash
python3 tools/import_daily_bars.py \
  --input samples/daily_bars.sample.csv \
  --output data/processed/daily_bars.csv \
  --metadata-output data/metadata/daily_bars.import.json
```

导入工具会执行：

- 必填字段检查。
- 交易日期校验。
- 股票代码格式校验。
- OHLC 价格关系校验。
- 成交量、成交额、换手率非负校验。
- 布尔字段标准化。
- 同一股票同一交易日重复记录检查。
- 标准 CSV 输出。
- 导入元数据输出。

## 4. 输出标准

标准输出 CSV 字段顺序固定为：

```text
trade_date,code,open,high,low,close,pre_close,volume,turnover,turnover_rate,is_limit_up,is_limit_down,is_suspended,adjust_type,data_source,updated_at
```

后续趋势强度策略和回测模块会读取 `data/processed/daily_bars.csv`。
