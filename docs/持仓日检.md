# 持仓日检

本文档定义持仓每日更新和风险检查流程。

持仓日检用于回答：

- 最新价格下是否触发止损。
- 是否接近止损。
- 当前收益和组合收益贡献是多少。
- 是否存在仓位上限风险。
- 观察项和备注是否持续留痕。

## 1. 输入

默认读取：

```text
positions/*.yaml
config/investment-profile.example.yaml
```

真实持仓记录默认不提交到 Git。

## 2. 更新内容

`tools/update_position_daily.py` 会更新：

- 当前价格。
- 当前收益率。
- 当前组合收益贡献。
- 持仓天数。
- 跟踪备注。

随后自动执行 `position_check.py` 的持仓风险检查。

## 3. 命令

```bash
python3 tools/update_position_daily.py \
  --position positions/POS-示例.yaml \
  --current-price 201 \
  --days-held 1 \
  --note "成交额继续放大，趋势仍在。" \
  --check-output data/metadata/POS-示例.daily-check.json
```

默认会原地更新 `--position`。如需输出到新文件：

```bash
python3 tools/update_position_daily.py \
  --position positions/POS-示例.yaml \
  --current-price 201 \
  --output /tmp/POS-示例.updated.yaml
```

## 4. 结论

日检结论沿用持仓风险检查：

- `normal`：正常。
- `warning`：有提醒，需要观察。
- `needs_action`：需要处理，例如触发止损。

当结论为 `needs_action` 时，命令退出码为 1。
